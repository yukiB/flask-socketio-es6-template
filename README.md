# flask-react-es6

Flask template with react.js es6.

## Requirements

- Python
  - 3.5.1 or greater
- Node.js
  - v7.2.1 or greater
- npm
  - 3.10.10 or greater


## Install

```
$ pip install -r requirements.txt
```

```
$ npm install
```

## Run

```
$ sh run.sh
```

To make changes, run gulp so it can compile them:

```
$ gulp
```

And access to `http:localhost:3001`
